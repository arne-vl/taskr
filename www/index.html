Command-Line Power, Simplified

taskr is a command-line runner that simplifies running complex tasks.

Installation

Installing taskr is done by downloading the binary of the latest release and placing it in a directory in your PATH.

Check the latest release here.

Help

$taskr -h Usage:   taskr <task_name> Options:   -h, --help    Show this help message and exit   -l, --list    List the available tasks

Task Definition

Here's a quick example of a taskr.toml file:

[build]
description = "build the project"
command = "make -C build"

[install]
command = "sudo ln -f bin/taskr /usr/bin/taskr"
dependencies = ["build"]
description = "link the taskr binary to /usr/bin/taskr"
